Про компанію

Компанія «ЕПАМ ДІДЖИТАЛ» є одним із лідерів у сфері IT, що надає послуги з розробки, тестування та консалтингу. Практика в такій компанії дозволила мені отримати безцінний досвід роботи в команді професіоналів, ознайомитися з корпоративними стандартами якості та сучасними підходами до організації робочого процесу.

Компанія EPAM має глобальну присутність у понад 30 країнах світу, де працює тисячі фахівців. Вона займається розробкою програмного забезпечення, тестуванням, консалтинговими послугами, а також впровадженням цифрових рішень для клієнтів. Практика у компанії такого рівня дозволила мені ознайомитися з високими стандартами роботи та корпоративною культурою.

Етап 2: Автоматизовані тести

Реалізовано автоматизовані тести для перевірки ключових функцій додатку, таких як авторизація, створення та редагування проектів, а також генерація звітів.

Показати реалізацію тесту -> 
    import org.openqa.selenium.By;
    import org.openqa.selenium.WebDriver;
    import org.openqa.selenium.WebElement;
    import org.openqa.selenium.chrome.ChromeDriver;
    
    public class LoginTest {
        public static void main(String[] args) {
            // Встановлення драйвера для Chrome
            System.setProperty("webdriver.chrome.driver", "path/to/chromedriver");
            WebDriver driver = new ChromeDriver();
    
            try {
                // Перехід на сторінку авторизації
                driver.get("https://example.com/login");
    
                // Введення логіну
                WebElement usernameField = driver.findElement(By.id("username"));
                usernameField.sendKeys("test_user");
    
                // Введення паролю
                WebElement passwordField = driver.findElement(By.id("password"));
                passwordField.sendKeys("password123");
    
                // Натискання кнопки входу
                WebElement loginButton = driver.findElement(By.id("loginButton"));
                loginButton.click();
    
                // Перевірка результату
                WebElement successMessage = driver.findElement(By.id("successMessage"));
                if (successMessage.isDisplayed()) {
                    System.out.println("Авторизація пройшла успішно!");
                } else {
                    System.out.println("Авторизація не вдалася.");
                }
            } finally {
                // Закриття браузера
                driver.quit();
            }
        }
    }

Етап 3: Інтеграція у процес CI/CD

Тести були інтегровані у процес CI/CD за допомогою Jenkins, що дозволило автоматично запускати їх після кожного оновлення коду.

Показати файл процесу -> 
                                pipeline {
                                    agent any
                                
                                    environment {
                                        CHROME_DRIVER_PATH = "/path/to/chromedriver" // Шлях до ChromeDriver
                                    }
                                
                                    stages {
                                        stage('Checkout') {
                                            steps {
                                                // Клонування репозиторію з автотестами
                                                git branch: 'main', url: 'https://github.com/ваш_репозиторій'
                                            }
                                        }
                                        stage('Install Dependencies') {
                                            steps {
                                                // Встановлення необхідних залежностей через Maven
                                                sh 'mvn clean install'
                                            }
                                        }
                                        stage('Run Tests') {
                                            steps {
                                                // Запуск тестів через Maven
                                                sh 'mvn test'
                                            }
                                        }
                                        stage('Generate Report') {
                                            steps {
                                                // Генерація звіту за допомогою Allure
                                                sh 'mvn allure:serve'
                                            }
                                        }
                                    }
                                
                                    post {
                                        always {
                                            // Збір артефактів (наприклад, звіти тестів)
                                            archiveArtifacts artifacts: '**/target/allure-results/**', allowEmptyArchive: true
                                        }
                                        success {
                                            echo 'Тести виконані успішно!'
                                        }
                                        failure {
                                            echo 'Тести завершились помилкою!'
                                        }
                                    }
                                }
